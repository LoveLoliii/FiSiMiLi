# JE_A
+ 此项目是使用Kotlin语言和mvvm架构进行编写的Android App。
+ 此项目使用了[自由神社曲谱库2.0](https://github.com/zytx121/je)提供的后台数据。
+ 此项目目前（2019/03/28）实现了通过搜索查看乐谱与简单播放的功能。
